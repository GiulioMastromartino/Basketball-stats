File Requirements

CSV Filename Format: Opponent_TeamScore-OppScore_DD-MM-YYYY_[F/S/P].csv

Examples:
• Lakers_95-88_15-03-2025_S.csv (Season game)
• Warriors_102-99_20-12-2024_F.csv (Friendly)
• Celtics_110-105_05-04-2025_P.csv (Playoff)

Game Type Codes (CSV):
  • F = Friendly
  • S = Season
  • P = Playoff
Required Stats Columns (CSV):

The CSV must include a row with Name='Total' (this row will be automatically skipped during import).

PDF Requirements:

The PDF should contain a box score table with the same columns as the CSV. If the header is not detected, fill the overrides (Opponent, Date, Scores).
