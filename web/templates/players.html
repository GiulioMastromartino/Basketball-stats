Statistics Glossary

Basic Stats

GP: Games Played
MIN: Minutes Per Game
PTS: Points Per Game
REB: Rebounds Per Game
OREB: Offensive Rebounds Per Game
DREB: Defensive Rebounds Per Game
AST: Assists Per Game
STL: Steals Per Game
BLK: Blocks Per Game
TOV: Turnovers Per Game
PF: Personal Fouls Per Game

Advanced Efficiency

ORtg: Offensive Rating - Points produced per 100 possessions
PPP: Points Per Possession - Scoring efficiency
USG: Usage - Average possessions used per game
EFF: Efficiency Rating - (PTS + REB + AST + STL + BLK) - (Missed FG + Missed FT + TOV)
AST/TOV: Assist-to-Turnover Ratio - Ball handling efficiency
CV%: Coefficient of Variation - Performance consistency (lower = more consistent)

Shooting Metrics

FG%: Field Goal Percentage
2P%: Two-Point Field Goal Percentage
3P%: Three-Point Field Goal Percentage
FT%: Free Throw Percentage
TS%: True Shooting % - Overall scoring efficiency (accounts for 3PT and FT)
eFG%: Effective Field Goal % - FG% adjusted for 3-pointers being worth more
FTA/FGA: Free Throw Attempt Rate - Aggression/contact rate
OREB%: Offensive Rebound % of total rebounds
Made/Att: Total shots made and attempted (shown below percentages)
